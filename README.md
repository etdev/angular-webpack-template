# angular-webpack-template
Complete template for front-end development

### What's Included
* Webpack
* AngularJS 1
  * Restangular
  * ngRoute
* Sass (via node-sass)
* Bourbon
* Neat
* Normalize.css
* Sane breakpoint defaults
* Sane typography defaults
* Lodash
* Underscore
* Fontawesome
* Raleway, Open Sans, Montseratt Google fonts
* slim/SLM template support
